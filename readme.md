## Usage

Make sure you have `Python >= 3.11` installed.


### Running the Project

1. Clone the repository:

   ```bash
   git clone https://github.com/Ant0in/Projet4-IA.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Projet4-IA/
   ```

3. Install poetry env:
   
   ```bash
   pip install poetry
   poetry install
   poetry shell
   poetry update
   ```

4. Run the project:

   ```bash
   python ".\main.py" (--options)
   ```
